# Memo Rewrite Lite

한국어 문장을 짧고 담백한 메모체로 바꿀 때 우선 적용하는 로컬 규칙.
지금은 아래 두 규칙만 우선 적용하고, 이후 필요할 때 항목을 추가하거나 수정하면 됨.

## 우선 규칙

### 1. 중립 어조 유지
- 단정형 서술은 완곡한 메모체 표현으로 낮춤.
- 예:
  - `~위험하다` -> `~위험해보임`
  - `~편이 맞다` -> `~보임`
  - 필요하면 `~것 같음` 형태 사용

### 2. 단문 메모체로 변환
- 문장을 짧게 끊고, 연결 표현과 수식어는 줄임.
- 아래 같은 종결을 우선 사용:
  - `~함`
  - `~됨`
  - `~봄`
  - `~보임`
  - `~해봄`
  - `~적합해보임`

## 출력 규칙
- 핵심 의미 유지
- 한국어만 출력
- 부가 설명 금지
- 과도한 통신체, 구어체, 불릿화는 피함
